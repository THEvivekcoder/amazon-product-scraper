# Amazon Product Scraper

A Python-based web scraping project that collects product information from Amazon search results using **Selenium** and **BeautifulSoup**.

The scraper extracts product **Title, Price, Rating, and Link** and stores the data in a structured **CSV file**.

---

## Project Overview

This project demonstrates how to combine **browser automation** and **HTML parsing** to build a simple data collection pipeline.

The workflow is divided into two steps:

1. **Scraping product HTML using Selenium**
2. **Extracting product data using BeautifulSoup**

---

## Features

- Automated browser scraping using Selenium
- Extracts product **Title**
- Extracts **Price**
- Extracts **Rating**
- Extracts **Product Link**
- Stores structured data in **CSV format**

---

## Tech Stack

- Python
- Selenium
- BeautifulSoup (bs4)
- CSV module
- HTML Parsing
